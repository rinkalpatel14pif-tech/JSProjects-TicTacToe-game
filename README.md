# Tic Tac Toe Game

A simple **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**.  
This project allows two players to play the classic Tic Tac Toe game in the browser. It is **fully responsive**, so you can play it on desktop, tablet, and mobile devices.
